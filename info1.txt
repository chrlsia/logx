Let's test all the new Phase 2 features:

```bash
# Level filter
cargo run -- read sample.log --level warn
```

```bash
# Inverted grep — show lines that do NOT contain "connection"
cargo run -- read sample.log --grep connection --invert
```

```bash
# Regex grep — lines containing "connection" OR "memory"
cargo run -- read sample.log --grep "connection|memory"
```

```bash
# Time filter — since 1 hour ago (all lines should show since sample is recent)
cargo run -- read sample.log --since 1h
```

